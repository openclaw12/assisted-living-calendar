# Assisted Living Calendar Platform

A full-stack scheduling platform for retirement communities. It captures resident preferences, auto-plans events with staff coverage, and delivers analytics, compliance logs, and a polished UI that's worth a subscription.

## Highlights
- Resident event polling + smart scheduling engine
- Staff shift coordination with gap alerts and prep checklists
- Personalized resident dashboards & engagement feeds
- Compliance logging, HIPAA-ready documentation, and audit trails
- Vendor/budget tracking plus executive analytics
- Responsive, high-contrast UI optimized for kiosks and tablets

## Getting Started
1. Clone the repo and install dependencies for both frontend and backend.
2. Populate `config/dev.env` with local secrets.
3. Run `docker-compose up` (or the dev scripts) to bring up the stack.
4. Use the scripts in `scripts/` to seed test data and generate monthly plans.

See `docs/requirements.md` for detailed scope and `docs/api-spec.md` for the REST contract.
